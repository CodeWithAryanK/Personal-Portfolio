# Aryan Kumar | Personal Portfolio

Welcome to the source code of **Aryan Kumar's Personal Portfolio** website.

---

## About

This is a modern, responsive personal portfolio website built to showcase my skills, projects, competitive programming profile, and contact information. The design features a cyberpunk-inspired aesthetic, with smooth animations and clean layouts, reflecting my passion as a B.Tech CSE student and full-stack developer.

---

## Features

- Clean and modern UI with glitch and gradient text effects
- Responsive navigation bar with quick access to different sections
- Detailed technical skills section highlighting expertise in C/C++, Java, and Web Development
- Project showcase with images, descriptions, technology tags, and GitHub links
- Competitive programming stats with direct link to my Codeforces profile
- Contact section with social media links and a fully functional contact form powered by Formspree
- SEO optimized with meta tags for better search engine visibility
- Social media Open Graph and Twitter card support for rich sharing previews

---

## Technologies Used

- **HTML5** for semantic markup
- **CSS3** (with external stylesheet) for styling and animations
- **JavaScript** (external `req.js`) for interactivity and form submission handling
- Responsive design techniques for cross-device compatibility
